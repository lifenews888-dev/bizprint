# BizPrint — Workflow Pipeline

Хэвлэлийн захиалгын Kanban pipeline систем.

## Суулгах

```bash
# 1. Фолдер руу орох
cd bizprint-workflow

# 2. Dependencies суулгах
npm install

# 3. Ажиллуулах
npm run dev
```

Браузер дээр `http://localhost:3000` нээгдэнэ.

## Боломжууд

- **5 шатлалт pipeline**: Шинэ → Дизайн → Бэлэн → Хэвлэж байна → Дууссан
- **Нэг товчоор төлөв шилжүүлэх**: Карт дээрх товч дарахад дараагийн шат руу шилжинэ
- **Audit trail**: Хэн, хэзээ, юу хийсэн бүгд бүртгэгдэнэ
- **Тэмдэглэл нэмэх**: Карт дарж modal нээгээд тэмдэглэл бичих
- **Хариуцагч хуваарилах**: Ажилтан assign хийх
- **Хэрэглэгч сонголт**: Дээрх dropdown-оор хэн ажиллаж байгааг сонгоно

## Build хийх

```bash
npm run build
```

`dist/` фолдерт бэлэн файлууд үүснэ.
