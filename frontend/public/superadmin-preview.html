  • Улаан "SA" avatar (бусад админ улбар шар "A")
  • "Супер Админ" гэсэн нэр (vs "Системийн Админ")
  • "Super Admin" subtitle BizPrint лого дор
  • "Систем (SA)" нэмэлт цэс бүлэг
  • Vendor Tier удирдлага
  • Ашиг тайлан (profit reports)
  • API Webhooks удирдлага
  • Систем тохиргоо

Энгийн Админ — Стандарт

  • "Системийн Админ" гэсэн нэр
  • "Admin Panel" subtitle
  • Хяналт, Захиалга, Бүтээгдэхүүн цэсүүд

Superadmin-аар нэвтрэх:

1. Backend серверийг ажиллуулна (npm run start:dev)
2. Admin Users хуудас руу очно (/admin/users)
3. Нэг хэрэглэгчийн role-г "superadmin" болгоно
4. Тэр имэйлээр нэвтрэхэд Супер Админ орчин харагдана
